## Installation

```bash
$ npm install

$ npm install g json-server 

# if the above json-server installation doesn't work(optional)
$ sudo npm install g json-server
```

## Running the app

```bash
# watch json data
$ npm run start:server

# watch nest rest api
$ npm run start:dev
```
